[иҜ‘] Oracle еҸ‘еёғдәҶ Secure Global Desktop Release 5.2

By K.X

еҸ‘иЎЁдәҺ 2015-05-08

ж–Үз« зӣ®еҪ•

жқҘжәҗ: http://www.dbta.com/Editorial/News-Flashes/Oracle-Rolls-Out-Secure-Global-Desktop-Release-52-103730.aspx

Oracleе·Із»ҸеҸ‘еёғдәҶOracle Secure Global Desktop (SGD) Release 5.2пјҢдҪңдёәиҜҘе…¬еҸёзҡ„иҝңзЁӢи®ҝй—®и§ЈеҶіж–№жЎҲпјҢдёәWindowsпјҢLinuxпјҢSolarisд»ҘеҸҠеӨ§еһӢдё»жңәзӯүжңҚеҠЎеҷЁз«Ҝжүҳз®Ўзҡ„дјҒдёҡзә§еә”з”ЁзЁӢеәҸе’ҢжЎҢйқўжҸҗдҫӣе®үе…Ёзҡ„иҝңзЁӢи®ҝй—®гҖӮ

SGDи®©з”ЁжҲ·иғҪеӨҹдҪҝз”Ёд»»дҪ•и®ҫеӨҮпјҢеңЁд»»дҪ•ең°ж–№йғҪеҸҜд»Ҙе®үе…Ёең°е·ҘдҪңпјҢеҗҢж—¶дёәиҝҗиЎҢеңЁж•°жҚ®дёӯеҝғзҡ„еә”з”ЁзЁӢеәҸе’ҢжЎҢйқўзҺҜеўғжҸҗдҫӣз®ЎзҗҶе‘ҳеҸҜжҺ§зҡ„и®ҝй—®пјҢеңЁBYODзҡ„дё–з•ҢдёӯдёәдјҒдёҡжҸҗдҫӣдәҶйқһеёёе…ій”®зҡ„дјҳеҠҝгҖӮ

SGDеҸҜд»Ҙе·ҘдҪңеңЁжөҒиЎҢзҡ„е®ўжҲ·з«Ҝи®ҫеӨҮдёҠпјҢеҢ…жӢ¬Windows PCгҖҒMacsгҖҒLinux PCд»ҘеҸҠиҜёеҰӮApple iPadе’ҢеҹәдәҺAndroidзҡ„е№іжқҝи®ҫеӨҮгҖӮ

е°Ҫз®Ўе…¶иҝҳжҳҜд»ҘеҺҹжқҘзҡ„вҖңTarantellaвҖқеҗҚеӯ—иў«дәәзҶҹзҹҘпјҢиҜҘдә§е“Ғзҡ„еҗҚз§°еңЁ2005е№ҙиў«Sun Microsystemsж”¶иҙӯж—¶ж”№жҲҗдәҶSun Secure Global DesktopпјҢд№ӢеҗҺеңЁOracleж”¶иҙӯSunд№ӢеҗҺеҸҲиў«жӣҙеҗҚдёәOracle Secure Global DesktopгҖӮ

вҖңе…¶е®һеңЁж”¶иҙӯд№ӢеүҚпјҢOracleе°ұдёҖзӣҙжҳҜиҜҘдә§е“Ғзҡ„жңҖеӨ§з”ЁжҲ·вҖқпјҢWim CoekaertsпјҢLinuxе’ҢиҷҡжӢҹеҢ–е·ҘзЁӢй«ҳзә§еүҜжҖ»иЈҒгҖӮеңЁTarantellaж—¶д»ЈпјҢOracleе°ұејҖе§ӢдҪҝз”Ёе®ғдәҶгҖӮвҖңжҲ‘д»¬е°Ҷе…¶з”ЁеңЁејҖеҸ‘зҺҜеўғдёӯпјҢдёәиҝңзЁӢејҖеҸ‘иҖ…жҸҗдҫӣжүҳз®ЎзҺҜеўғзҡ„и®ҝй—®пјҢеҗҢж—¶иҝңзЁӢз”ЁжҲ·д№ҹйҖҡиҝҮе®ғжқҘи®ҝй—®йӣҶдёӯз®ЎзҗҶзҡ„еә”з”ЁзЁӢеәҸзҺҜеўғвҖқпјҢд»–еҰӮжҳҜиҜҙгҖӮ

еҸҰеӨ–пјҢOracleжң¬иә«е°ұжңүжҲҗзҷҫдёҠеҚғзҡ„зҡ„еә”з”ЁзЁӢеәҸпјҢиҝҷдәӣеә”з”ЁзЁӢеәҸеӨ§еӨҡж•°жҳҜеҹәдәҺWebжҠҖжңҜзҡ„пјҢCoekaertsжҸҗеҲ°гҖӮеҪ“дёҚеҗҢзүҲжң¬зҡ„еә”з”ЁзЁӢеәҸеҜ№жөҸи§ҲеҷЁзҡ„йңҖжұӮд№ҹдёҚеҗҢзҡ„ж—¶еҖҷпјҢSGDдјҡеёҰжқҘжҳҫи‘—зҡ„дјҳеҠҝгҖӮ

вҖңеҸҜд»ҘжҳҜд»»дҪ•зұ»еһӢзҡ„е®ўжҲ·з«ҜпјҢеӣ дёәе®ғжҳҜеҹәдәҺжөҸи§ҲеҷЁзҡ„гҖӮе®ғдҪҝз”ЁдәҶHTML5пјҢеӣ жӯӨдҪ еҸҜд»ҘдҪҝз”ЁiPadпјҢжҲ–иҖ…Androidи®ҫеӨҮпјҢжҲ–иҖ…дҪ зҡ„Mac - иҝҷзңҹзҡ„дёҚйҮҚиҰҒвҖқпјҢCoekaertsиҜҙпјҢвҖңеә”з”ЁзЁӢеәҸзҡ„дҪҝз”ЁдҪ“йӘҢе°ұе’Ңжң¬ең°еә”з”ЁдёҖжЁЎдёҖж ·пјҢиҖҢдё”еҸҜд»ҘеңЁд»»дҪ•зҪ‘з»ңзҺҜеўғдёӢдҪҝз”ЁвҖқпјҢиҖҢдё”пјҢд»Һе®үе…ЁжҖ§зҡ„и§’еәҰжқҘиҜҙпјҢеӣ дёәжң¬ең°жІЎжңүеӯҳеӮЁж•°жҚ®пјҢжүҖд»ҘиҝҷжҳҜдёҖз§Қи®ҝй—®дјҒдёҡж•°жҚ®зҡ„йқһеёёе®үе…Ёзҡ„ж–№жі•гҖӮ

жӯӨеӨ–пјҢиҜҙеҲ°е®ғиғҪжҸҗдҫӣзҡ„еҠҹиғҪпјҢCoekaertsиҜҙпјҢз”ЁжҲ·еҸҜиғҪдјҡдҪҝз”ЁжҹҗдёӘзүҲжң¬зҡ„еә”з”ЁзЁӢеәҸпјҢиҖҢиҜҘеә”з”ЁзЁӢеәҸйңҖиҰҒInternet Explorer 6пјҢеҗҢж—¶д»–иҝҳеҸҜиғҪжңүдёҖдёӘж–°зүҲжң¬зҡ„еә”з”ЁзЁӢеәҸйңҖиҰҒIE7пјҢеҸҜй—®йўҳжҳҜеңЁдёҖдёӘжЎҢйқўзҺҜеўғдёӯеҸӘиғҪиҝҗиЎҢдёҖдёӘзүҲжң¬зҡ„IEгҖӮдҪҶжҳҜпјҢеңЁжүҳз®ЎзҺҜеўғдёӯпјҢеҸҜд»Ҙе®үиЈ…еӨҡдёӘдёҚеҗҢзүҲжң¬зҡ„жөҸи§ҲеҷЁгҖӮвҖңиҝҷжҳҜдёҖдёӘе…ёеһӢзҡ„дҫӢеӯҗпјҢиҜҒжҳҺдәҶеңЁзңҹе®һдё–з•Ңдёӯзҡ„е®ўжҲ·зҺҜеўғдёӯSGDйқһеёёжңүз”ЁгҖӮвҖқ

SGD Release 5.2жҸҗдҫӣдёҖдәӣйҮҚиҰҒзҡ„еҠҹиғҪејәеҢ–пјҢCoekaertsжҸҗеҲ°гҖӮ

дёҺOracle Access ManagerйӣҶжҲҗзҡ„еҚ•зӮ№зҷ»еҪ•(Single sign-on) иғҪеӨҹеҜ№еә”з”ЁзЁӢеәҸе®һзҺ°еҚіж—¶и®ҝй—®пјҢиҖҢз”ЁжҲ·еҸӘйңҖиҰҒиҫ“е…ҘдёҖж¬Ўиә«д»ҪдҝЎжҒҜгҖӮвҖңеҸҜд»Ҙиҝҷд№ҲиҜҙпјҢжҲ‘д»¬е®һйҷ…дёҠд»ҺдёҖиҝӣй—ЁејҖе§Ӣе°ұйӣҶжҲҗдәҶеҚ•зӮ№зҷ»еҪ•вҖқпјҢCoekaertsиҜҙгҖӮOracle Secure Global DesktopдёҺOracle Access Managerзҡ„йӣҶжҲҗеҸҜд»ҘдҪҝз”ЁOracle Access Managerзҡ„еҚ•зӮ№зҷ»еҪ•зҡ„еҠҹиғҪжқҘи®ӨиҜҒSGDжңҚеҠЎеҷЁзҡ„з”ЁжҲ·пјҢе’Ңд»ҺSGDе·ҘдҪңеҢәжү“ејҖзҡ„иҝңзЁӢеә”з”ЁзЁӢеәҸзҡ„з”ЁжҲ·гҖӮеҚ•зӮ№зҷ»еҪ•дҪҝSGDз”ЁжҲ·еңЁи®ҝй—®еӨҡдёӘеә”з”ЁзЁӢеәҸж—¶еҸӘйңҖиҰҒзҷ»еҪ•дёҖж¬ЎпјҢз®ҖеҢ–дәҶеҜҶз Ғз®ЎзҗҶгҖӮ

еҲ©з”Ё Oracle Enterprise Manager зӣ‘и§ҶпјҢеңЁ5.2еҸ‘иЎҢзүҲдёӯйҖҡиҝҮжҸ’д»¶жқҘе®һзҺ°зҡ„пјҢеңЁOracle Enterprise ManagerдёӯеҜ№еӨҡдёӘSGDйғЁзҪІзҺҜеўғиҝӣиЎҢйӣҶдёӯзҡ„зӣ‘и§ҶгҖӮвҖңOracle Enterprise ManagerжҳҜOracleзҡ„дёҖз«ҷејҸз®ЎзҗҶеҘ—д»¶пјҢе®ғиғҪеӨҹз®ЎзҗҶж•°жҚ®еә“пјҢдёӯй—ҙд»¶пјҢеә”з”ЁзЁӢеәҸпјҢеӯҳеӮЁи®ҫеӨҮпјҢLinuxзӯүзӯүгҖӮOracleзҡ„жүҖжңүдә§е“ҒйғҪдјҡдёәиҝҷдёӘйӣҶдёӯз®ЎзҗҶе·Ҙе…·жҸҗдҫӣдёҖдёӘжҸ’д»¶пјҢиҝҷе°ұж„Ҹе‘ізқҖеңЁзӣ‘и§Ҷе’Ңз®ЎзҗҶжүҖжңүдёҚеҗҢзҡ„з»„д»¶ж—¶пјҢдҪ еҸӘйңҖиҰҒеңЁиҝҷдёҖдёӘе·Ҙе…·дёӯеҠ е…ҘдёҖдёӘжҸ’д»¶еҚіеҸҜвҖқпјҢCoekaertsиҜҙпјҢвҖңеңЁ5.2дёӯпјҢжҲ‘д»¬зҺ°еңЁж”ҜжҢҒе°ҶSGDдҪңдёәEnterprise Managerзҡ„дёҖдёӘз»„д»¶жҲ–иҖ…дёҖдёӘжҸ’д»¶вҖқгҖӮ

йҖҡиҝҮиҝҷдәӣжҸ’д»¶пјҢеҸҜд»Ҙзӣ‘и§Ҷе’ҢеҲҶжһҗSGDжңҚеҠЎеҷЁд»ҘеҸҠGatewayжңҚеҠЎеҷЁзҡ„еҒҘеә·е’ҢжҖ§иғҪзҠ¶жҖҒгҖӮеҜ№дәҺSGDжңҚеҠЎеҷЁпјҢиҜҘжҸ’д»¶ж”¶йӣҶеҗ„з§ҚжҖ§иғҪе’Ңй…ҚзҪ®жҢҮж ҮпјҢиҜёеҰӮйҳөеҲ—зҠ¶жҖҒпјҢз”ЁжҲ·е’Ңеә”з”ЁзЁӢеәҸпјҢж•°жҚ®еӯҳеӮЁеҜ№иұЎпјҢSGDжңҚеҠЎеҷЁиҝӣзЁӢпјҢзүҲжң¬д»ҘеҸҠиЎҘдёҒзӯүгҖӮеҜ№дәҺGatewayжңҚеҠЎеҷЁпјҢеҗ„з§ҚжҖ§иғҪе’Ңй…ҚзҪ®жҢҮж Үд№ҹдјҡиў«ж”¶йӣҶпјҢеҢ…жӢ¬жңҚеҠЎдёӯзҡ„иҝһжҺҘдҝЎжҒҜпјҢжіЁеҶҢзҡ„SGDжңҚеҠЎеҷЁпјҢзүҲжң¬пјҢиЎҘдёҒзӯүгҖӮиҝҷдәӣжҸ’д»¶ж”¶йӣҶзҡ„жҢҮж Үж•°жҚ®еҸҜд»Ҙиў«дёҖдәӣжҠҘиЎЁе·Ҙе…·пјҢеҰӮOracle Business Intelligence PublisherиҝӣиЎҢиҝӣдёҖжӯҘзҡ„еҲҶжһҗе’ҢеӨ„зҗҶгҖӮ

еңЁж–°еҸ‘иЎҢзүҲдёӯзҡ„з¬¬дёүдёӘз»„д»¶жҳҜеўһејәзҡ„еҹәдәҺи§’иүІзҡ„з”ЁжҲ·з®ЎзҗҶжңәиғҪпјҢCoekaertsиҜҙпјҢйҡҸзқҖдә‘и®Ўз®—е’Ңжүҳз®ЎзҺҜеўғзҡ„дёҚж–ӯеўһй•ҝпјҢиҝҷжҳҜи¶ҠжқҘи¶ҠйҮҚиҰҒзҡ„еҠҹиғҪгҖӮвҖңдј з»ҹдёҠпјҢдёҖдёӘдә§е“ҒеҸӘйңҖиҰҒдёҖдёӘз®ЎзҗҶе‘ҳжқҘз®ЎзҗҶжүҖжңүдёҚеҗҢзҡ„з”ЁжҲ·вҖқгҖӮдҪҶжҳҜпјҢOracleе·Із»Ҹе°Ҷз®ЎзҗҶе‘ҳд№ҹеҲҶжҲҗдёҚеҗҢзҡ„ж°ҙе№іе’ҢжқғйҷҗгҖӮеңЁиҝҷдёҖзүҲжң¬дёӯеј•е…Ҙзҡ„ж–°е‘Ҫд»Өд№ҹз®ҖеҢ–дәҶSGDйғЁзҪІдёӯиҪҜд»¶жӣҙж–°пјҲиЎҘдёҒпјүзҡ„е®үиЈ…иҝҮзЁӢгҖӮйҷӨдәҶдҪҝз”Ёе‘Ҫд»ӨиЎҢпјҢеӣҫеҪўеҢ–з•Ңйқўзҡ„з®ЎзҗҶжҺ§еҲ¶еҸ°еҸҜд»Ҙз”ЁдәҺж·»еҠ SGDжңҚеҠЎеҷЁеҲ°йҳөеҲ—дёӯпјҢеҗҢж—¶дҪҝз”Ёе®үе…Ёзҡ„йҳөеҲ—еҶ…йҖҡдҝЎпјҢвҖңеңЁдә‘зҺҜеўғдёӯпјҢеңЁжүҳз®Ўдё»жңәжңҚеҠЎзҺҜеўғдёӯпјҢиҝҷдјҡжҳҜдёҖдёӘеҫҲдёҚй”ҷзҡ„еҠ еҲҶзӮ№вҖқгҖӮ

ж–°зүҲжң¬дёӯзҡ„PulseAudioйӣҶжҲҗеҸҜд»ҘеҜ№дҪҝз”ЁPulseAudioзҡ„еә”з”ЁзЁӢеәҸе®һзҺ°йҹійў‘йҮҚе®ҡеҗ‘пјҢеӨ§еӨ§еўһеҠ дәҶSGDж”ҜжҢҒзҡ„йҹійў‘еә”з”ЁзЁӢеәҸиҢғеӣҙгҖӮжӯӨеӨ–пјҢж–°еўһеҠ зҡ„еҜ№UNIXйҹійў‘иҫ“е…Ҙзҡ„ж”ҜжҢҒпјҢдҪҝд№ӢдёҺеҜ№Windowsзҡ„йҹійў‘ж”ҜжҢҒеӨ„дәҺеҗҢдёҖж°ҙе№ізәҝдёҠпјҢиҖҢдё”д№ҹеҸҜд»ҘдёәдёҺOSS,SADAжҲ–PulseAudioе…је®№зҡ„еә”з”ЁзЁӢеәҸдҪҝз”Ёйҹійў‘иҫ“е…ҘгҖӮ

йҷӨдәҶж–°зү№жҖ§е’Ңжӣҙе№ҝжіӣзҡ„жңҚеҠЎеҷЁпјҢе®ўжҲ·з«ҜпјҢд»ҘеҸҠжөҸи§ҲеҷЁж”ҜжҢҒд№ӢеӨ–пјҢжң¬ж¬ЎеҸ‘иЎҢзүҲд№ҹжҸҗдҫӣдәҶеўһејәзҡ„е®үе…ЁжҖ§е’ҢжҖ§иғҪпјҢеңЁеҗ„з§ҚзҪ‘з»ңзҺҜеўғдёӢжҸҗдҫӣжӣҙдё°еҜҢзҡ„з”ЁжҲ·дҪ“йӘҢгҖӮ

Oracle Secure Global Desktop 5.2з§үжҢҒе…¶дёҖиҙҜзҡ„з®ҖжҙҒжҖ§еҗҢж—¶жӣҙзҙ§еҜҶең°дёҺOracleдә§е“ҒпјҢжҠҖжңҜд»ҘеҸҠи§ЈеҶіж–№жЎҲиҝӣиЎҢдәҶйӣҶжҲҗпјҢеҰӮOracle Managed Cloud ServicesпјҢOracle Enterprise ManagerпјҢе’ҢOracle Access ManagerгҖӮ
