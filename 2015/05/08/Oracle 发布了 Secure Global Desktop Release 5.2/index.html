вАЬеЕґеЃЮеЬ®жФґиі≠дєЛеЙНпЉМOracleе∞±дЄАзЫіжШѓиѓ•дЇІеУБзЪДжЬАе§ІзФ®жИЈвАЭпЉМWim CoekaertsпЉМLinuxеТМиЩЪжЛЯеМЦеЈ•з®ЛйЂШзЇІеЙѓжАїи£БгАВеЬ®TarantellaжЧґдї£пЉМOracleе∞±еЉАеІЛдљњзФ®еЃГдЇЖгАВвАЬжИСдїђе∞ЖеЕґзФ®еЬ®еЉАеПСзОѓеҐГдЄ≠пЉМдЄЇињЬз®ЛеЉАеПСиАЕжПРдЊЫжЙШзЃ°зОѓеҐГзЪДиЃњйЧЃпЉМеРМжЧґињЬз®ЛзФ®жИЈдєЯйАЪињЗеЃГжЭ•иЃњйЧЃйЫЖдЄ≠зЃ°зРЖзЪДеЇФзФ®з®ЛеЇПзОѓеҐГвАЭпЉМдїЦе¶ВжШѓиѓігАВ

еП¶е§ЦпЉМOracleжЬђиЇЂе∞±жЬЙжИРзЩЊдЄКеНГзЪДзЪДеЇФзФ®з®ЛеЇПпЉМињЩдЇЫеЇФзФ®з®ЛеЇПе§Іе§ЪжХ∞жШѓеЯЇдЇОWebжКАжЬѓзЪДпЉМCoekaertsжПРеИ∞гАВељУдЄНеРМзЙИжЬђзЪДеЇФзФ®з®ЛеЇПеѓєжµПиІИеЩ®зЪДйЬАж±ВдєЯдЄНеРМзЪДжЧґеАЩпЉМSGDдЉЪеЄ¶жЭ•жШЊиСЧзЪДдЉШеКњгАВ

вАЬеПѓдї•жШѓдїїдљХз±їеЮЛзЪДеЃҐжИЈзЂѓпЉМеЫ†дЄЇеЃГжШѓеЯЇдЇОжµПиІИеЩ®зЪДгАВеЃГдљњзФ®дЇЖHTML5пЉМеЫ†ж≠§дљ†еПѓдї•дљњзФ®iPadпЉМжИЦиАЕAndroidиЃЊе§ЗпЉМжИЦиАЕдљ†зЪДMac - ињЩзЬЯзЪДдЄНйЗНи¶БвАЭпЉМCoekaertsиѓіпЉМвАЬеЇФзФ®з®ЛеЇПзЪДдљњзФ®дљУй™Ме∞±еТМжЬђеЬ∞еЇФзФ®дЄАж®°дЄАж†ЈпЉМиАМдЄФеПѓдї•еЬ®дїїдљХзљСзїЬзОѓеҐГдЄЛдљњзФ®вАЭпЉМиАМдЄФпЉМдїОеЃЙеЕ®жАІзЪДиІТеЇ¶жЭ•иѓіпЉМеЫ†дЄЇжЬђеЬ∞ж≤°жЬЙе≠ШеВ®жХ∞жНЃпЉМжЙАдї•ињЩжШѓдЄАзІНиЃњйЧЃдЉБдЄЪжХ∞жНЃзЪДйЭЮеЄЄеЃЙеЕ®зЪДжЦєж≥ХгАВ

ж≠§е§ЦпЉМиѓіеИ∞еЃГиГљжПРдЊЫзЪДеКЯиГљпЉМCoekaertsиѓіпЉМзФ®жИЈеПѓиГљдЉЪдљњзФ®жЯРдЄ™зЙИжЬђзЪДеЇФзФ®з®ЛеЇПпЉМиАМиѓ•еЇФзФ®з®ЛеЇПйЬАи¶БInternet Explorer 6пЉМеРМжЧґдїЦињШеПѓиГљжЬЙдЄАдЄ™жЦ∞зЙИжЬђзЪДеЇФзФ®з®ЛеЇПйЬАи¶БIE7пЉМеПѓйЧЃйҐШжШѓеЬ®дЄАдЄ™ж°МйЭҐзОѓеҐГдЄ≠еП™иГљињРи°МдЄАдЄ™зЙИжЬђзЪДIEгАВдљЖжШѓпЉМеЬ®жЙШзЃ°зОѓеҐГдЄ≠пЉМеПѓдї•еЃЙи£Ее§ЪдЄ™дЄНеРМзЙИжЬђзЪДжµПиІИеЩ®гАВвАЬињЩжШѓдЄАдЄ™еЕЄеЮЛзЪДдЊЛе≠РпЉМиѓБжШОдЇЖеЬ®зЬЯеЃЮдЄЦзХМдЄ≠зЪДеЃҐжИЈзОѓеҐГдЄ≠SGDйЭЮеЄЄжЬЙзФ®гАВвАЭ

SGD Release 5.2жПРдЊЫдЄАдЇЫйЗНи¶БзЪДеКЯиГљеЉЇеМЦпЉМCoekaertsжПРеИ∞гАВ
