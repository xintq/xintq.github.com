  • 主服务器： P2（10.194.153.9）
  • 从服务器： P22（10.194.153.10）

在主服务器（P2）上创建集群

p2:~# pvecm create abc #创建群集名称

在从服务器上加入到集群

p22:~# pvecm add 10.194.153.9     #输入集群所在的主服务器的IP
